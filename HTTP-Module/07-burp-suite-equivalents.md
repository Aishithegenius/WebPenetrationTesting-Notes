BURP SUITE EQUIVALENTS

Browser DevTools:
- View requests
- Modify headers
- Replay requests

Burp Proxy:
- Intercepts traffic
- Modifies requests live

cURL:
- Manual request sending

Burp Repeater:
- Same as cURL but interactive
- Re-send modified requests

Burp Intruder:
- Automated attacks
- Similar to scripted cURL loops
