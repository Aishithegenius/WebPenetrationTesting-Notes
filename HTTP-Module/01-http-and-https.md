HYPERTEXT TRANSFER PROTOCOL (HTTP)

HTTP is a protocol used for communication between a client and a server.

The client sends an HTTP request.
The server sends back an HTTP response.

HTTP properties:
- Stateless
- Text-based
- Used to transfer HTML, JSON, images, scripts

HYPERTEXT TRANSFER PROTOCOL SECURE (HTTPS)

HTTPS is HTTP with encryption.

HTTPS:
- Uses TLS/SSL
- Encrypts traffic
- Prevents sniffing and man-in-the-middle attacks

HTTP sends data in plain text.
HTTPS encrypts all communication.

Common ports:
HTTP uses port 80
HTTPS uses port 443
